FAQ

Questions answered.

Does TokenWatch read my Claude conversations?

No. TokenWatch only reads UI-level signals — banner text, status indicators, and activity presence — never conversation content. Your prompts and Claude's responses are completely private.

How does it detect usage without reading conversation content?

The extension watches the claude.ai DOM for usage-limit banners and activity signals. For precise percentages, the optional Claude Code CLI integration reads from your local CLI — no conversation data ever leaves your machine.

What does the CLI link actually do?

Running npx tokenwatch-link init installs a small script that reads your Claude CLI usage percentage and syncs it to TokenWatch via a secure pairing code. No credentials stored in plain text.

Does it work with all Claude plans?

Yes. DOM detection works on any Claude plan. The CLI integration works with Claude Code (all tiers). Most useful for teams actively using Claude Pro or Claude Code daily.

Can I try paid features before buying?

Yes. All paid plans include a 14-day free trial, no credit card required. If you don't upgrade before it ends, you move to Free automatically — nothing is deleted.

What happens if my team grows past the plan cap?

New invitations are blocked with an upgrade prompt. Existing teammates are never removed. No surprise charges, ever. Upgrade when you're ready.
