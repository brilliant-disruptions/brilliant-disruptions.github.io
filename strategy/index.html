04 · HOLES TO WATCH

Where the line
can jam.

Speed only holds if we keep these from stalling the queue.

Feedback loop blocking

If Nick's marketing needs iteration from the technical side, the build queue can stall.

Messy handoffs

Nick will have product questions and feature requests that pull Michael + Jimmy back in.

Hidden coupling

Shared infrastructure — auth, backend, troubleshooting time — turns parallel builds into friction.
